이어폰 전용 타이머 Earphone Timer

: :

이 웹앱에 대하여 / About this web app

이 타이머는 당신이 조용한 공간에서 사용할 때 이어폰에서만 알람 소리가 들리도록 합니다.
이어폰을 연결하지 않았을 때 스피커에서는 알람 소리가 매우 작게 들릴 수 있으며, 조용한 환경에서만 잘 들릴 수 있습니다.
이어폰을 꼭 착용하고 사용하는 것을 권장합니다.
원하는 시간을 설정하고 시작 버튼을 누르면 설정한 시간 후에 알람이 울립니다.
중지 및 초기화 버튼으로 타이머를 조작할 수 있습니다.
현재 시각은 당신 기기의 시간 기준입니다.

This timer ensures the alarm sound only plays through your earphones when used in a quiet space.
If earphones are not connected, the alarm may be very quiet from the speakers and audible only in silent environments.
We recommend always using earphones.
Set the desired time and press the start button; the alarm will sound after the set duration.
Use stop and reset buttons to control the timer.
The current time is based on your device’s clock.
